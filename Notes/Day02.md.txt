# Day 02 - Variables and Data Types

## What is a Variable?

A variable is a named memory location used to store data that can change during program execution.

Example:

```csharp
int age = 25;
```

---

## Data Types

### int

Stores whole numbers.

Example:

```csharp
int age = 25;
```

---

### double

Stores decimal numbers.

Example:

```csharp
double height = 170.5;
```

---

### decimal

Stores high-precision decimal values.

Used for:
- Banking
- Financial applications
- Money calculations

Example:

```csharp
decimal salary = 25000.50m;
```

---

### bool

Stores:

- true
- false

Example:

```csharp
bool isPassed = true;
```

---

### char

Stores a single character.

Example:

```csharp
char grade = 'A';
```

---

### string

Stores text.

Example:

```csharp
string name = "Pavan";
```

---

## Difference between int and double

| int | double |
|------|---------|
| Whole numbers | Decimal numbers |
| Example: 10 | Example: 10.5 |

---

## Difference between char and string

| char | string |
|------|---------|
| Single character | Multiple characters |
| 'A' | "Pavan" |

---

## Console.ReadLine()

Reads user input from the keyboard.

Returns:

string

Example:

```csharp
string name = Console.ReadLine();
```
