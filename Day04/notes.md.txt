# What is a Loop?

A loop is used to execute a block of code repeatedly until a condition becomes false.

Instead of writing the same code multiple times, we use loops.

Example

```csharp
for(int i = 1; i <= 5; i++)
{
    Console.WriteLine(i);
}
```

---

# Types of Loops

## 1. for Loop

Used when the number of iterations is known.

Syntax

```csharp
for(initialization; condition; increment)
{
    // Code
}
```

Example

```csharp
for(int i = 1; i <= 5; i++)
{
    Console.WriteLine(i);
}
```

Output

```
1
2
3
4
5
```

---

## 2. while Loop

Used when the number of iterations is unknown.

Example

```csharp
int i = 1;

while(i <= 5)
{
    Console.WriteLine(i);
    i++;
}
```

---

## 3. do...while Loop

Executes the code at least once before checking the condition.

Example

```csharp
int i = 10;

do
{
    Console.WriteLine(i);
}
while(i < 5);
```

Output

```
10
```

---

## 4. foreach Loop

Used to iterate through collections like arrays and lists.

Example

```csharp
List<string> names = new List<string>()
{
    "Pavan",
    "Ravi",
    "Kiran"
};

foreach(string name in names)
{
    Console.WriteLine(name);
}
```

---

# break Statement

Stops the loop immediately.

Example

```csharp
for(int i = 1; i <= 10; i++)
{
    if(i == 5)
        break;

    Console.WriteLine(i);
}
```

Output

```
1
2
3
4
```

---

# continue Statement

Skips the current iteration.

Example

```csharp
for(int i = 1; i <= 5; i++)
{
    if(i == 3)
        continue;

    Console.WriteLine(i);
}
```

Output

```
1
2
4
5
```

---

# Nested Loop

A loop inside another loop.

Example

```csharp
for(int i = 1; i <= 3; i++)
{
    for(int j = 1; j <= i; j++)
    {
        Console.Write("*");
    }

    Console.WriteLine();
}
```

Output

```
*
**
***
```

# Difference Between Console.Write() and Console.WriteLine()

| Console.Write() | Console.WriteLine() |
|-----------------|---------------------|
| Prints output | Prints output |
| Cursor remains on same line | Cursor moves to next line |
| Used for user prompts | Used for displaying output |

Example

```csharp
Console.Write("Enter Name: ");
string? name = Console.ReadLine();
```

Output

```
Enter Name: Pavan
```

---

# Difference Between Console.Read() and Console.ReadLine()

| Console.Read() | Console.ReadLine() |
|----------------|--------------------|
| Reads one character | Reads entire line |
| Returns ASCII/Unicode integer value | Returns string |
| Rarely used | Mostly used in real projects |

Example

```csharp
char ch = (char)Console.Read();
```

Input

```
A
```

Output

```
A
```

---

# Difference Between for and foreach

| for | foreach |
|------|----------|
| Uses index | No index |
| Best when index is needed | Best for reading collections |
| Can modify using index | Used mainly for iteration |

# Common Mistakes

## Infinite Loop

Wrong

```csharp
for(int i = 1; i <= 5;)
```

Forgot

```
i++
```

---

## Wrong Loop Condition

Wrong

```csharp
for(int i = 0; i <= count; i++)
```

Correct

```csharp
for(int i = 0; i < count; i++)
```

---

## Modifying Collection in foreach

Wrong

```csharp
foreach(var item in list)
{
    item = "ABC";
}
```

Use a `for` loop or create a new collection when modification is needed.

# Interview Questions and Answers

## 1. What is a loop?

Answer

A loop executes a block of code repeatedly until a condition becomes false.

---

## 2. Difference between for and foreach?

Answer

- for uses an index.
- foreach iterates through each item in a collection.
- foreach is easier to read.

---

## 3. Difference between while and do...while?

Answer

while checks the condition before execution.

do...while executes once before checking the condition.

---

## 4. What is break?

Answer

Stops the loop immediately.

---

## 5. What is continue?

Answer

Skips the current iteration and continues with the next iteration.

---

## 6. Difference between Console.Write() and Console.WriteLine()?

Answer

Console.Write()

- Prints output.
- Cursor remains on the same line.

Console.WriteLine()

- Prints output.
- Cursor moves to the next line.

---

## 7. Difference between Console.Read() and Console.ReadLine()?

Answer

Console.Read()

- Reads one character.
- Returns an integer (ASCII/Unicode value).

Console.ReadLine()

- Reads the entire line.
- Returns a string.

---

## 8. Why is foreach preferred?

Answer

foreach is cleaner, easier to read, and reduces indexing mistakes when simply reading items from a collection.

# Key Points to Remember

- Use `for` when the number of iterations is known.
- Use `while` when the condition controls the loop.
- `do...while` executes at least once.
- Use `foreach` for collections.
- `break` exits the loop.
- `continue` skips the current iteration.
- Use `Console.Write()` for prompts.
- Use `Console.WriteLine()` for displaying output.
- Use `Console.ReadLine()` for user input.
