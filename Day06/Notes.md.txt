# What is an Array?

An array is a collection of elements of the **same data type** stored in **contiguous memory locations**.

Example

```csharp
int[] numbers = { 10, 20, 30, 40, 50 };
```

---

# Why Do We Need Arrays?

Without Arrays

```csharp
string doc1 = "Invoice.pdf";
string doc2 = "Resume.pdf";
string doc3 = "OfferLetter.pdf";
string doc4 = "Salary.xlsx";
```

If there are 1000 documents, creating 1000 variables is impossible.

Using an Array

```csharp
string[] documents = new string[1000];
```

One variable can store many values.

# Array Index

Arrays start from **0**.

Example

| Index | Value |
|-------|-------|
| 0 | 10 |
| 1 | 20 |
| 2 | 30 |
| 3 | 40 |
| 4 | 50 |

Access

```csharp
Console.WriteLine(numbers[0]);
```

Output

```
10
```

---

# Why Does Array Index Start at 0?

Arrays start at index **0** because the index represents the **memory offset** from the first element.

The first element is 0 positions away from the starting memory location.

# Loop Through an Array

Using `for`

```csharp
for(int i = 0; i < numbers.Length; i++)
{
    Console.WriteLine(numbers[i]);
}
```

---

# foreach Loop

```csharp
foreach(int number in numbers)
{
    Console.WriteLine(number);
}
```

Use `foreach` when you only need to read values.

# Common Exceptions

## IndexOutOfRangeException

Wrong

```csharp
Console.WriteLine(numbers[5]);
```

If the array has only 5 elements, valid indexes are

```
0
1
2
3
4
```

Accessing index 5 throws

```
IndexOutOfRangeException
```

# Difference Between for and foreach

| for | foreach |
|------|----------|
| Uses index | No index |
| Can update values using index | Best for reading values |

# Interview Questions and Answers

## 1. What is an Array?

Answer

An array is a collection of elements of the same data type stored in contiguous memory locations.

---

## 2. Why do arrays start from index 0?

Answer

Because the index represents the memory offset from the first element.

---

## 3. Can arrays store different data types?

Answer

No.

All elements must be of the same data type.

---

## 4. What is the difference between Array and List?

Array

- Fixed size

List

- Dynamic size

---

## 5. What is Length?

Answer

Length returns the total number of elements in an array.

---

## 6. What is IndexOutOfRangeException?

Answer

It occurs when we try to access an index that does not exist in the array.

---

## 7. What are the default values of arrays?

| Data Type | Default Value |
|------------|---------------|
| int | 0 |
| double | 0 |
| decimal | 0 |
| bool | false |
| string | null |

# Key Points to Remember

- Arrays store elements of the same data type.
- Arrays have a fixed size.
- Index starts at 0.
- Use `Length` to get the array size.
- Use `for` when index is needed.
- Use `foreach` when only reading values.
- Accessing an invalid index throws `IndexOutOfRangeException`.
